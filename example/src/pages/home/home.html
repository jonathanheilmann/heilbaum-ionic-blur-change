blurChange directive

heilbaum-ionic-blur-change

heilbaum-ionic-blur-change provides an Ionic 2 directive to detect changes on input blur and emits an event.

Example

  • Fill or modify input:
    -> blurChange will be emitted.
  • Focus input and blur without doing changes:
    -> blurChange will not be emitted.
  • Focus input, do some changes, discard changes and blur:
    -> blurChange will not be emitted.

Usage

Add (blurChange) attribute to ion-input component assigning a method to be called on blurChange.
(blurChange) passes an object $event of type BlurChangeEvent containing properties oldValue and newValue.
